# estimand-missing-val-diagn-stud
Framework for estimands and missing values in diagnostic accuracy studies

This repository belongs to the manuscript entitled "Joint Framework of the Estimand Concept and Missing Values in Diagnostic Accuracy Studies – a Simulation Study" by Stahlmann, K., Fierenz, A., Badpa, M. and Zapf, A. (Institute of Medical Biometry and Epidemiology, University Medical Center Hamburg Eppendorf, Hamburg, Germany). It provides the simulation protocol, which was developed before the conduct of the final simulation study, as well as the related final programs and simulated data upon which the results of the manuscript are based on.  

The simulation was conducted on the high performance cluster “Hummel-2” operated by the University of Hamburg and running on Debian GNU/Linux. The simulated data were analysed on a local machine. R version 4.4.2 or higher were used for simulating and analysing the data.
