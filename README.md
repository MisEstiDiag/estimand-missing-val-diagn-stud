# estimand-missing-val-diagn-stud
Framework for estimands and missing values in diagnostic accuracy studies

This project belongs to the joint framework of estimands and missing values in diagnostic accuracy studies. As this project is currently being worked on, this repository provides the simulation protocol for the planned simulation study. The simulation programs will be available in this repository when the simulation study has been conducted and the respective manuscript has been submitted.
